Please follow below guidelines to run this sample.
[1] Deploy mygoods data file in a server and create a API out of it from the API Manager publisher
[2] Login to the store and create a custom application and subscribe the newly added API to the app
[3] Generate consumer key and consumer secret
[4] Please replace the consumer key in the index.html file client_id and edit the oauth_gateway, api_url to match with your
 configurations
 [5] Deploy the sample in any server and play with it
